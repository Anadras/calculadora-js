# 🧮 Calculadora JavaScript

Uma calculadora funcional construída com **JavaScript puro**, sem frameworks ou bibliotecas externas. O foco do projeto é a prática sólida de conceitos fundamentais do JS: manipulação do DOM, eventos e lógica de programação.

## ✨ Funcionalidades

- Operações básicas: adição, subtração, multiplicação e divisão
- Limpeza do display (C) e apagamento do último dígito (⌫)
- Encadeamento de operações em sequência
- Prevenção de divisão por zero
- Interface responsiva e intuitiva

---

## 🛠️ Tecnologias utilizadas

| Tecnologia        | Uso                                       |
| ----------------- | ----------------------------------------- |
| HTML5             | Estrutura da interface e botões           |
| CSS3              | Estilização e layout da calculadora       |
| JavaScript (ES6+) | Lógica das operações e manipulação do DOM |

## 💡 Conceitos praticados

Este projeto foi desenvolvido com foco em aprendizado. Os principais conceitos aplicados foram:

- **Manipulação do DOM** — seleção de elementos com `querySelector` e atualização dinâmica do display
- **Event Listeners** — captura de cliques nos botões com `addEventListener`
- **Lógica de programação** — controle de estado da operação atual, operandos e resultado
- **Funções** — organização do código em funções reutilizáveis e com responsabilidade única
- **Tratamento de casos especiais** — divisão por zero, múltiplos operadores, entradas inválidas
